# 🌱 스프링 입문

코드로 배우는 **Spring Boot**, **웹 MVC**, 그리고 **DB 접근 기술**을 다룹니다.  
초보 개발자부터 실무자까지, 스프링의 핵심 개념을 쉽게 이해할 수 있도록 구성된 프로젝트입니다.

---

## 🚀 기술 스택 (Tech Stack)
- **Framework**: Spring Boot  
- **Language**: Java  
- **Build Tool**: Gradle  

---

## 📚 프로젝트 구성
1. **Spring Boot 설정**  
   스프링 부트의 기본 설정과 Gradle 기반 프로젝트 빌드 구조를 학습합니다.

2. **웹 MVC**  
   컨트롤러와 뷰를 연결하는 MVC 패턴 구현을 배웁니다.

3. **DB 접근 기술**  
   스프링 데이터 JPA를 활용하여 데이터베이스와 상호작용하는 방법을 익힙니다.
